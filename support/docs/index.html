Browsing Servers

Search System

The global search bar allows you to instantly find servers by name, description keywords, or tags. Search results update in real time, providing fast and accurate results.

  • Search by server name
  • Search by keywords in descriptions
  • Search by category or interest tags

Filtering & Sorting

Advanced filtering tools allow you to narrow results by multiple tags, language, community size, and activity level.

  • Multi-tag filtering
  • Member count sorting
  • Newest and trending listings
  • Activity-based ranking

NSFW Safety Controls

NSFW listings are hidden by default and can only be enabled by verified adult users. Minors cannot view NSFW listings under any circumstances.

  • Strict age-gated visibility
  • Manual NSFW content toggle
  • Automated detection and review

Submitting a Server

Required Information

Each server submission must include accurate and complete information to ensure a smooth review process.

  • Server name
  • Permanent Discord invite link
  • Server description
  • Server logo (high quality recommended)
  • Relevant tags
  • Discord server ID

Optional Fields

  • Official website
  • Server rules
  • NSFW designation

Review & Approval Process

All submissions undergo manual review by OpsLink staff to verify accuracy, legitimacy, and compliance.

  • Content verification
  • Rule compliance review
  • Manual quality inspection
  • Approval or denial notification

Editing Server Information

Important Policy

All server edits must be requested through the official support ticket system. Direct editing is not available.

Why Tickets Are Required

  • Prevents abuse and unauthorized changes
  • Ensures consistent quality control
  • Allows manual verification of ownership
  • Maintains audit trails for accountability

How to Request Edits

  • Open a support ticket
  • Provide server ID and invite link
  • Clearly describe requested changes
  • Wait for staff review and confirmation

User Accounts

Authentication

OpsLink uses Discord OAuth for secure and seamless authentication. No passwords are stored on our servers.

User Capabilities

  • Submit servers
  • Submit support tickets
  • Report rule violations
  • Leave community feedback

Law Enforcement & Legal Requests

Official Contact Channel

All law enforcement inquiries, legal requests, and emergency disclosures must be submitted through our official support ticket system. This ensures proper identity verification, request tracking, and legal compliance.

How to Contact OpsLink

  • Open a support ticket through the Support page
  • Select the category: Law Enforcement Request
  • Provide official agency details and case reference numbers
  • Include a clear and specific description of the request

Request Handling

  • All requests are reviewed by authorized OpsLink compliance staff
  • Requests are logged and audited for legal accountability
  • Emergency requests are prioritized
  • All disclosures follow applicable laws and privacy regulations

Support & Tickets

When to Open a Ticket

  • Server edit requests
  • Bug reports
  • Account issues
  • Content appeals

Response Times

Most tickets are answered within 24–48 hours. Complex requests may take longer.

Frequently Asked Questions

How long does approval take?
Most submissions are reviewed within 24–72 hours.

Can I edit my server myself?
No. All edits must be submitted through the support ticket system.

Why was my server denied?
Servers may be denied for rule violations, missing data, misleading descriptions, or inappropriate content.

How do I report a server?
Click the report button on any listing and submit your reason.
