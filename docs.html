Browsing Servers

Use the search bar to find servers by name, description, or tags. Filters allow combining multiple tags to refine results.

Submitting Servers

Server Name, Invite Link, Description, Logo URL, Tags, Discord Server ID, Optional: Website, Rules, NSFW toggle.
Servers are pending review by OpsLink admins. Notifications are sent upon approval or denial.
Users can edit server information via their account page. Updates are reviewed again for approval.

Server Modals

Shows server logo, name, description, tags, owner info, and join button.
Users can leave reviews, comments, and report inappropriate servers directly from the modal.

User Accounts

Accounts are created via Discord OAuth. Username and role are visible in navbar.
Regular users can submit servers and comments. Admins have dashboard access for moderation.

Admin Dashboard

Review, approve, deny, or delete server submissions and reports.
All actions are logged and sent to a Discord webhook for accountability.

Server Tags

Tags describe server topics. New tags are auto-created when submitted if not existing.
Users can click tags to filter server lists.

Support & Tickets

Users submit tickets for help or questions via the Support page. Admins respond directly.
Admins track, respond, and close tickets while maintaining a record of support history.

Reporting Servers

Click “Report” on a server modal and provide a reason for admins to review.
Admins can delete servers violating rules based on reports.
