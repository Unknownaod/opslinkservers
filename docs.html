OpsLink Docs

Welcome to OpsLink documentation! This guide explains every feature of the site so you can browse, submit, and manage Discord servers effectively.

Browsing Servers

The Browse Servers page lets you explore all Discord servers submitted to OpsLink.

  • Search: Use the search bar to find servers by name, description, or tags.
  • Filter by Tag: Click on any tag to only show servers with that tag. You can also select multiple tags.
  • NSFW Toggle: Check this box to show servers marked as NSFW (Not Safe For Work).

Each server card shows its logo, description, tags, member count, and a link to join the server. Clicking a card opens a detailed modal with more info and reviews.

Submitting Servers

To submit your Discord server, go to the Submit page. You'll need:

  • Server Name
  • Server Invite Link (Discord.gg link)
  • Description (what your server is about)
  • Server Logo URL
  • Tags (to help users find your server)
  • Discord Server ID (for member tracking via our bot)
  • Optional: Website, Rules, NSFW toggle, Server Type, Language

Once submitted, your server will be pending approval by the OpsLink team. You will receive a notification when it is approved or denied.

Server Modals

Clicking a server card opens a detailed modal containing:

  • Server Logo and Name
  • Description and Tags
  • Server Owner Info
  • Join Server button (opens Discord invite)
  • Reviews and Comments from other users
  • Option to report the server if inappropriate

User Accounts

Creating an account allows you to submit servers, post comments, and report servers.

  • Login is done via Discord OAuth for easy verification.
  • Once logged in, you can see your username and role (user/admin) in the navbar.
  • Admins get access to the Admin Dashboard for managing server submissions.

Admin Dashboard

Admins have a dashboard to review, approve, deny, or delete server submissions.

  • View all pending, approved, and denied servers
  • Approve or deny servers with a reason
  • Delete inappropriate servers
  • View reports submitted by users

Admin actions are logged and notifications are sent to a Discord webhook for transparency.

Server Tags

Tags are keywords that describe a server's focus, such as Gaming, Art, Tech, or Community.

  • Users can filter servers by tags to find what interests them.
  • Tags must be unique per server and are limited in length.
  • New tags are automatically created when a server submits one that doesn’t exist.

Support

OpsLink provides support through:

  • Docs: This page contains explanations for all site features.
  • Support Tickets: Users can submit tickets for help, questions, or issues.

Tickets are handled by OpsLink staff, and responses are sent back to the user’s account.

Reporting Servers

Users can report servers that are inappropriate, spam, or violate Discord's rules.

  • Reports are visible to admins in the dashboard.
  • Admins can delete servers based on reports.
  • Users are encouraged to provide a clear reason when reporting.
